## 功能列表

- 首页

- 闲置页面,能够发表和联系卖家

- 快递页面,能够详细地查看快递信息

- 互帮页面,能够解决给校园内的各种事情(约人跑步,借书等等)

- 闲置物品搜索功能

- 个人页面,能够通过校园卡与手机号码进行注册(提供短信验证)
